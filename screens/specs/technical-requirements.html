Technical Requirements

Software, libraries, and environment for running the system
Development Environment build & code
  • Java JDK 8 or higher (JDK 17 recommended)
  • NetBeans IDE 8.2 or later
  • XAMPP 7.4+ (includes Apache + MySQL / MariaDB)
  • phpMyAdmin (bundled with XAMPP)
  • Git for version control
  • Operating System: Windows 10+ / macOS / Linux
Runtime Requirements execution
  • Java Runtime Environment (JRE) 8 or higher
  • MySQL 5.7+ or MariaDB 10.4+
  • Minimum 4 GB RAM
  • 200 MB disk space (application + database)
  • Display resolution: 1280×720 or higher
  • Network: localhost only (no internet required)
Libraries & Dependencies third-party JARs
  • mysql-connector-java-5.1.23-bin.jar
    MySQL JDBC Driver (provided in NetBeans)
  • jbcrypt-0.4.jar
    Password hashing (BCrypt)
  • jfreechart-1.5.x.jar + jcommon-1.0.24.jar
    Charting library for Reports & Charts panel
  • Optional: jasperreports-6.x.x.jar
    PDF report generation (future feature)
Database Configuration connection
  • Database name: inventory_db
  • Host: localhost (port 3306, default)
  • Default credentials: root / (empty password)
  • Character set: utf8mb4_unicode_ci
  • JDBC URL:
    jdbc:mysql://localhost:3306/inventory_db
  • Schema: 14 tables · 4 views
  • Constraints: 19 foreign keys · 24 CHECK constraints
SETUP STEPS: (1) Install XAMPP and start Apache + MySQL services · (2) Open phpMyAdmin → create database inventory_db → Import inventory_schema_final.sql · (3) Open NetBeans project → Libraries → Add MySQL JDBC Driver, jbcrypt, jfreechart · (4) Run project → Login with admin / admin123
