Quick Start Tutorial

A typical day in the system, in 6 steps
01 Sign in LoginFrame
Enter your username and password, then click Login. The main window opens to the Dashboard.
02 Check the Dashboard DashboardPanel
Review today's KPIs, low-stock alerts, and recent transactions. Use Quick Actions to jump anywhere.
03 Record a Sale SalesPanel
Add items + quantities to the invoice, then click Save Transaction. Stock goes down automatically.
04 Record a Delivery ReceivingPanel
Pick the vendor, list what arrived, click Save Receipt. Stock goes up automatically.
05 Check Stock On Hand StockOnHandPanel
View live quantities for every item. Critical and low items are highlighted in red / amber.
06 End-of-day Audit AuditPanel
Walk the storage, type the physical count in the blue cells. The system flags any variance vs system stock.
TIP: Every action saves to a single MySQL database, so every panel always shows the latest data — no manual refresh, no syncing between screens.
